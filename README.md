## Giới thiệu

Đây là dự án website thương mại điện tử cho thương hiệu nước hoa Gray Collection, được xây dựng bằng React + Vite + TypeScript.

Dự án tập trung vào trải nghiệm mua sắm hiện đại với product catalog, wishlist, cart, checkout, trang hồ sơ người dùng và các luồng thanh toán thực tế.

## Tính năng chính

- Trang chủ và collection page
- Trang chi tiết sản phẩm
- Gift set page
- Wishlist và shopping cart
- Đăng nhập và hồ sơ khách hàng
- Checkout flow
- Trang trả kết quả thanh toán MoMo / VNPay
- Policy pages và các thành phần storefront hoàn chỉnh
- API service layer để kết nối dữ liệu sản phẩm và tài khoản

## Cấu trúc ứng dụng

- `src/components/`: các UI components
- `src/context/`: auth, cart, wishlist
- `src/pages/`: các trang chính của website
- `src/services/api.ts`: lớp gọi API sản phẩm, khách hàng, checkout
- `src/types/`: định nghĩa type cho account và perfume
